# `notes/donina-emonyms/` — per-emonym extractions from Donina (2016)

One file per emonym that Donina works through in §§2.1.* of her thesis.
Each file is an English-language extraction of the cryptoclass profile
Donina builds for that emonym across the 20 national variants of English,
on the corpora NOW, GloWbE and (for validation) COCA.

These files serve as the **direct comparator** for the corresponding
Spanish emonym's Phase 1 / Phase 6 profile in this project. Concretely:
when the Spanish *miedo* profile is built in Phase 1, the answer to
"is this finding cross-linguistically robust?" depends on comparing
against `fear.md` row-by-row.

Source-page numbers refer to the printed page = PDF page in
`references/Диссертация_Донина_О.В..pdf` (no offset). For verification
practice see `notes/README.md`.

Per `notes/README.md`: if a file in this directory disagrees with the
PDF, the PDF wins. These extractions paraphrase and condense; for any
numeric value (per-cent shares, example counts, correlation
coefficients) or any verbatim quotation that will end up in scholarly
writing, verify against the source page before citing.
